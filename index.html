Gravel Travel

A gritty travel wishlist

A solo project and my very first full stack app. Gravel Travel is a travel wishlist app where a user can track countries and locations they would like to visit and log trips to locations they have visited.

Initially completed over just a single week a third of the way into my course at CodeClan. I decided to return to Gravel Travel after graduating as a chance to revisit Python & SQL, apply the knowledge I had gained during the remiainder of the course to overhaul the app as well as to add gain some completely new skills.

The project was a chance to work on object oriented programming with Python, web programming using the model-view-controller design pattern, build RESTful routes and perform CRUD actions with a postgreSQL database.

Tech stack: Python, Flask, Jinja, PostgreSQL, CSS, JavaScript

AeroSure?

Predicting weather-based delays for your flight

A full stack React.js app using the Open-Meteo's free Weather Forecast and Geocoding APIs. Produced in a single week as part of a three person group project.

AeroSure? is a weather delay prediction tool. The user enters their flight's details (i.e. departure and arrival locations and times). The locations are entered as stings and converted into geographic coordinates using the geocoding API, these coordinates are then fed into the forecast API which returns hourly weather forecasts at each location. Logic is then performed which determines if adverse weather conditions (such as thunder, hail or heavy winds) near the departure and arrival times could cause a delay to the flight. A graded traffic light system is used to output a simple prediction to the user.

This was my first app created as part of a team and was a perfect opportuniy to explore agile working methods. Some of the practices we adopted to improve productivity and cohesion were daily stand ups, mob programming and a final project retrospective.

Integrating two separate API requests into the logic of the app was a big learning curve and significantly improved my confidence and practical ability of using APIs.

Tech stack: JavaScript, React.js, MongoDB, Express.js, HTML, CSS

Abstract

The joy of pixels

A React.js app which draws abstract images via randomly generated coloured brushstrokes. A solo project.

This small app began as a test to see if I could generate abstract images in real time. The first version used divs and unsurprisingly the browser significantly slowed down once several hundred divs had been generated. Looking for an alternative solution I discovered the canvas element and refactored the code but at the same time tried to keep the aesthetic and feel of the original div based styling.

I later added some controlls for interactivity that allow the user to change several input parameters such as the size of the generated dots or the frequency of colour cycling.

Tech stack: JavaScript, React.js, HTML

Film Wishlist

Rate your films and find new favourites

Full-stack single-page React.js app for rating films and saving films to a wish list. Generates and outputs stats based on those ratings (such as most watched actors or highest rated genre) which are used to recommend new films to discover. A pair project completed in a two week timeframe.

This was my first exposure to using external component libraries. Material UI was used for the form elements. The design and particularly the way the user navigates between pages is a major feature and it behaves in an unusual way to most web apps.

Lots of practice with Git branches and merging.

Tech stack: JavaScript, React.js, MongoDB, Express.js, HTML, CSS
